# 정통 명리 사주·운세 안내

정통 명리학의 원리를 바탕으로 사주풀이, 토정비결, 궁합, 이름풀이, 택일을 제공하는 웹 서비스입니다.  
이 문서는 **이 사이트를 이용하시는 분들**을 위한 간단한 안내입니다.

---

## 제공 서비스

- **평생 사주풀이**
  - 태어난 연·월·일·시를 바탕으로 평생의 큰 운의 흐름, 재물운, 직업운, 성향과 기질을 풀이합니다.

- **토정비결 (연간 운세)**
  - 특정 연도를 선택하면 그 해의 전체 운세와 상·하반기 흐름, 월별 주의점 등을 풀이합니다.

- **궁합 보기**
  - 두 사람의 사주를 비교하여 성향의 조화, 상생·상극 관계, 함께할 때의 장단점을 풀이합니다.

- **이름풀이**
  - 이름(한글/한자)이 사주의 부족한 기운을 어떻게 보완하는지, 성명학 관점에서 풀이합니다.

- **택일 / 이사 날짜 보기**
  - 이사, 결혼, 개업 등 중요한 일을 앞두고 사주에 맞는 길일(좋은 날)을 안내합니다.

---

## 이용 방법

1. **메뉴 선택**
   - 화면 상단의 메뉴에서 “평생사주 / 토정비결 / 궁합보기 / 이름풀이 / 택일/이사” 중 원하는 항목을 선택합니다.

2. **기본 정보 입력**
   - 이름 (닉네임 입력 가능)
   - 성별
   - 양력/음력(평달/윤달)
   - 태어난 연·월·일·시  
   - (궁합의 경우) 상대방의 위 정보를 추가로 입력
   - (이름풀이의 경우) 해당 이름의 한자(알고 있다면) 입력

3. **“사주 분석하기” 또는 메뉴에 맞는 버튼 클릭**
   - 버튼을 누르면 잠시 후 분석이 완료되고, 결과가 화면 하단에 나타납니다.

4. **어려운 용어가 많을 때**
   - 결과 상단의 **“어려운 용어 AI 간소화”** 버튼을 누르면  
     명리학의 전문 용어를 보다 쉬운 현대어로 풀어 쓴 버전으로 바뀝니다.
   - 간소화 모드에서는 한자 표기가 자동으로 제거되고, 이해하기 쉬운 한글 설명 위주로 보여집니다.
   - 다시 버튼을 누르면 원래의 전문 용어 버전으로 돌아갑니다.

---

## 이용 시 유의사항

- **학술적 참고용 서비스**
  - 이 서비스는 명리학 이론과 통계를 바탕으로 운세를 풀이하지만,  
    어디까지나 **참고용 안내**이며, 실제 인생의 모든 결정을 대신해 줄 수는 없습니다.

- **의학·법률·투자 등 전문 자문이 아님**
  - 건강, 법률, 재테크 등은 반드시 각 분야의 전문 상담(의사, 변호사, 재무 전문가 등)을 우선으로 하시기 바랍니다.

- **개인 정보**
  - 이 서비스는 생년월일시 정보를 바탕으로 풀이를 생성합니다.
  - 민감한 추가 개인정보(주민등록번호, 상세 주소, 계좌번호 등)는 입력하지 마십시오.
  - 입력하신 정보는 사주 풀이 생성을 위해 일시적으로 서버(Cloudflare Worker 및 Grok API)를 거치지만, 별도의 회원 가입이나 데이터베이스 저장은 하지 않습니다.

## 자주 묻는 질문 (FAQ)

- **Q. 태어난 시간을 모르면 어떻게 하나요?**  
  A. “모름(낮 12시로 산정)”을 선택하시면, 일반적인 기준 시각(오시)을 바탕으로 풀이합니다.

- **Q. 결과 내용이 너무 어렵게 느껴집니다.**  
  A. 결과 상단의 **“어려운 용어 AI 간소화”** 버튼을 눌러 보십시오.  
  전문 용어를 보다 풀어서 설명한 버전으로 다시 보여 드립니다.

- **Q. 실제 사주 상담과 같나요?**  
  A. 명리학 원리에 기반하여 진지하게 풀이되지만,  
  직접 대면 상담에서 얻는 세부 상담과는 차이가 있을 수 있습니다.  
  중요한 결정은 여러 의견을 함께 고려하여 신중히 판단해 주시기 바랍니다.
